# Interactive-Dashboards-on-Power-BI
This project consists of an interactive sales dashboard on Power BI that utilizes essential visualisation to effectively communicate business performance insights of company X

**Key Features**
* Analysis of data that defines KPIs, employee distribution across different departments, salary and performance ratings. 
* Visualisations through different chart types for specific data insights
* An inetractive dashboard that integrates multiple visulaisations to cohesively depict business performance
